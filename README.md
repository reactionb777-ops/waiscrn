# WaiScrn

Скриншотер для Windows от [Waicore](https://waicore.com). Живёт в трее: жмёшь
PrintScreen, выделяешь область, по желанию рисуешь поверх - и одним движением
кидаешь в буфер, сохраняешь на диск или заливаешь в сеть. Если залил - ссылка
сразу лежит в буфере, бери и вставляй. Заливка идёт через
[imgdb.io](https://imgdb.io).


## Что умеет

- Выделение области сразу по всем мониторам. Рядом с курсором - пиксельная
  лупа (15×15), чтобы попадать ровно в нужный пиксель, и координаты.
- Рисование поверх снимка: карандаш, линия, стрелка, гибкая (от руки) стрелка,
  прямоугольник, текст. Палитра из шести цветов.
- Текст по-нормальному: многострочный (Shift+Enter), таскается мышкой,
  выделяется (Ctrl+A, двойной клик по слову), Ctrl+C / Ctrl+V / Ctrl+X,
  навигация стрелками / Home / End.
- Четыре режима выдачи, переключаются в трее:
  - в буфер обмена;
  - сохранить PNG в «Снимки экрана» и заодно копировать;
  - сохранить и залить - в буфер уходит ссылка;
  - только залить - в буфер уходит ссылка.
- Кнопка загрузки прямо в тулбаре: включил на один скрин - он уйдёт ссылкой,
  не трогая выбранный режим.
- Заливка на imgdb.io, ссылка по умолчанию живёт 72 часа.
- Стартует вместе с Windows (включено по умолчанию, выключается в трее).
- Сам проверяет обновления при запуске и тихо ставит новую версию.

Снимки сохраняются в `%USERPROFILE%\Pictures\Screenshots`.

## Как пользоваться

Жмёшь PrintScreen — экран замирает, сверху появляется тулбар, курсор
становится крестиком. Дальше:

- Просто выделил область и отпустил — снимок сразу уходит по текущему режиму,
  без лишних кнопок.
- Хочешь подрисовать — выбери инструмент, нарисуй, потом Enter или зелёную
  галочку.
- Esc или правый клик — отмена. Ctrl+Z — убрать последнюю фигуру.

## Хоткеи

| Клавиши | Действие |
| --- | --- |
| PrintScreen | снять в текущем режиме (выбран в трее) |
| Shift + PrintScreen | принудительно сохранить и копировать |
| Ctrl + PrintScreen | принудительно только в буфер |
| Ctrl + Shift + S | запасной хоткей, если PrintScreen занят системой |
| Enter | подтвердить |
| Esc / правый клик | отмена |
| Ctrl + Z | отменить последнюю фигуру |

PrintScreen иногда перехватывают «Ножницы» или OneDrive. Если так — программа
сама садится на Ctrl+Shift+S и пишет об этом.

## Установка

Скачай `WaiScrn-Setup.exe` из
[релизов](https://github.com/reactionb777-ops/waiscrn/releases) и запусти.
Программа поставится в Program Files, добавится в автозапуск и дальше будет
обновляться сама.

Нужны права администратора (выскочит запрос UAC) — без них не получится ловить
горячие клавиши поверх других окон и снимать защищённые окна.

Удаляется как обычная программа — через «Установка и удаление программ».

## Системные требования

Windows 10 или 11, 64-битная.
